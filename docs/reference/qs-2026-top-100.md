# QS 2026 Top 100 Reference

This project uses `data/rankings/qs-world-university-rankings-2026-top-100.json` as the canonical crawl-order reference for QS World University Rankings 2026.

Source: https://www.topuniversities.com/qs-top-uni-wur

Important batching rule:

- Use `rankText` / `rankNumber` for QS rank ranges.
- Use `rowNumber` only as source-list order.
- When a rank is tied, include every university with that `rankNumber`.

Example: QS ranks 31-35 are:

- 31: King's College London
- =32: Australian National University (ANU)
- =32: The Chinese University of Hong Kong (CUHK)
- 34: The University of Edinburgh
- 35: The University of Manchester

Do not substitute row-order batches or memory-derived batches for this file.
