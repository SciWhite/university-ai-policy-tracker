# Reference Sheet Knowledge Index

## Purpose

This directory contains retrieval-friendly summaries of human-created
spreadsheets and other benchmark references. They are useful for recall checks,
coverage comparisons, and crawl planning.

## Authority Boundary

Authority level: non_authoritative_benchmark.

Reference sheets are not official evidence. They cannot create canonical
claims, provide public source attribution, or replace official source snapshots.
Use them only to ask follow-up questions such as "should OpenClaw search this
institution/source again?"

## Current Files

- `plsc-edtechai-policy-v4-summary.md` - workbook-level summary and benchmark
  interpretation.
- `plsc-edtechai-policy-v4-columns.md` - sheet and column dictionary.

## Required Metadata For Converted Sheets

Each converted reference file should include:

- original file path
- source file name
- sheet names
- selected sheet or workbook-wide scope
- row count
- column count
- SHA-256 checksum
- conversion timestamp
- converter/runtime
- limitations and non-authoritative warning

## Do Not Use For

- Public JSON.
- Official source lists.
- Claim text.
- Evidence snippets.
- Review decisions.
